# 🚀 Lead Generation Platform

A modern full-stack lead generation platform that enables users to capture, manage, and store customer inquiries efficiently. The application provides a responsive landing page with a dynamic lead capture form, secure backend APIs, and a scalable database for managing potential customer information.

---

# 📌 Overview

The Lead Generation Platform is designed to help businesses collect high-quality leads through an intuitive web interface. Submitted leads are validated, securely stored in a database, and can be used for follow-up sales or marketing campaigns.

The project demonstrates full-stack web development using modern technologies, API integration, responsive UI design, and database management.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/shivm2003/lead_genration.git
```

```bash
cd lead_genration
```

---

## Install Backend Dependencies

```bash
cd server
npm install
```

---

## Install Frontend Dependencies

```bash
cd ../client
npm install
```

---

# 🔧 Environment Variables

Create a `.env` file inside the `server` directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

FRONTEND_URL=http://localhost:3000
```

---

# ▶️ Running the Application

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```

Application URLs:

```text
Frontend
http://localhost:3000

Backend
http://localhost:5000
```
